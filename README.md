# Organizador Automático de Arquivos em Python

Script em Python que organiza automaticamente uma pasta cheia de arquivos bagunçados.

O programa identifica o tipo de cada arquivo, cria pastas específicas (Imagens, PDFs, Documentos, Planilhas, Áudios e Vídeos) e move tudo para o lugar correto, renomeando os arquivos de forma padronizada.

## Funcionalidades

- Criação automática de pastas por tipo
- Movimentação automática dos arquivos
- Renomeação padronizada (IMG_001, PDF_001, DOC_001...)
- Funciona em qualquer pasta do computador
- Não utiliza bibliotecas externas

## Como usar

1. Coloque o `organizador.py` dentro da pasta que deseja organizar
2. Abra o terminal nessa pasta
3. Execute:

4. A organização acontece automaticamente

## Tecnologias

- Python
- pathlib
- shutil

## Objetivo

Projeto de automação para resolver tarefas repetitivas do dia a dia de forma simples e eficiente.
