# Organizador Automático de Pastas em Python

Este projeto é um script em Python criado para resolver um problema muito comum: pastas cheias de arquivos desorganizados.

O script identifica o tipo de cada arquivo, cria pastas automaticamente (Imagens, PDFs, Documentos, Planilhas, Áudios, Vídeos) e move os arquivos para o local correto, além de renomear todos de forma padronizada.

## Funcionalidades

- Cria pastas automaticamente por tipo de arquivo
- Move os arquivos para a pasta correta
- Renomeia os arquivos em sequência (IMG_001, PDF_001, DOC_001, etc.)
- Funciona em qualquer pasta do computador
- Não precisa instalar nenhuma biblioteca externa
